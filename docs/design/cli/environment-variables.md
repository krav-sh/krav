# Environment variables

Environment variables provide defaults for common options. All environment variables use the `KRAV_` prefix.

## Variable reference

| Variable | Description |
|---|---|
| `KRAV_SOCKET` | Sets the Unix socket path for server communication. |
| `KRAV_URL` | Sets the server HTTP URL. |
| `KRAV_LOG_LEVEL` | Sets the log level. |
| `KRAV_PROJECT` | Sets the project root. |
| `KRAV_CONFIG_FILE` | Overrides the configuration path. |
| `KRAV_NO_CONFIG` | Disables configuration loading entirely when set. |
